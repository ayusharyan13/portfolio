E-Commerce App

Built by Ayush Aryan

Overview

A full-stack e-commerce marketplace application built with Flutter for the frontend and Node.js + Express + MongoDB for the backend. Features user authentication, product management, shopping cart, and order processing.

Tech Stack

Flutter Node.js Express MongoDB Mongoose REST API Provider (State Management)

Key Features

  • Cross-platform mobile app working on both Android and iOS
  • RESTful API architecture with Express and MongoDB using Mongoose ODM
  • State management using Flutter's Provider pattern
  • User authentication and session management
  • Product catalog with search and filtering
  • Shopping cart and order management system
