Covid-19 Tracker App

Built by Ayush Aryan

Overview

A real-time COVID-19 tracking application built with Flutter and REST API integration. Provides global and country-specific statistics with interactive visualizations, safety guidelines, and real-time updates powered by reliable data sources.

Tech Stack

Flutter Dart REST API Postman Data Visualization

Key Features

  • Real-time COVID-19 statistics with global and country-specific data
  • Interactive charts for visualizing trends and comparisons
  • Country flags and region-based filtering
  • Safety guidelines and health recommendations
  • Cross-platform support for Android and iOS
  • Responsive UI with clean, accessible design
