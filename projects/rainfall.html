Rainfall Prediction App

Built by Ayush Aryan

Overview

A machine learning project that predicts rainfall occurrence using the Australia Weather dataset. The model applies data preprocessing techniques and classification algorithms to deliver accurate rainfall predictions through an intuitive user interface.

Tech Stack

Python Machine Learning Scikit-learn Pandas NumPy Data Preprocessing Model Deployment

Key Features

  • Rainfall prediction using the Australia Weather dataset with real-world meteorological features
  • Data preprocessing pipeline including handling missing values, feature encoding, and scaling
  • Multiple classification models evaluated for optimal prediction accuracy
  • User-friendly interface for inputting weather parameters and getting predictions
  • Model performance metrics and visualization of prediction results
